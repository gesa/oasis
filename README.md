# To the oasis!

## Running the server, building the site

### Dependencies:
- Ruby—see `.ruby-version`
- Node—see `package.json`

### Install:

```bash
npm install
```

### Run:

```bash
npm start
```

### Test:

("test" is a generous description of this function)

```bash
npm test
```

### Deploy:

Deploy to S3 using travis by running `travis setup s3` on the command line
